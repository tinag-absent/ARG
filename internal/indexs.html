海蝕機関に関わるな

本ポータルは、日本国内における海蝕現象の監視・収束に従事する海蝕員のための内部サイトです。 部外者は直ちに接続を終了してください。

海蝕機関とは

海蝕機関は、日本国内で発生する「海蝕現象」の観測・記録・収束を任務とする非公開組織です。 公的機関の表向きの記録には一切残されません。

主な任務: 海蝕現象の早期検知 / 現場封鎖 / 収束作業 / 証拠隠蔽 / 住民記憶処理（必要時）。

日報ログ

日付を指定して過去の海蝕現象収束記録を検索できます。 一部の高危険度事案は権限レベルにより閲覧制限がかかっています。

例: 「2023-11-02」「2024-08-13-港湾部事案」など。 特定のキーワードでのみ出現する記録が含まれている可能性があります。

機関部門一覧

Division-01

収束部門

主任務: 現場出動 / 海蝕現象の直接収束 / モジュール運用。 想定脅威度: 中〜極高。

海蝕現象の前線に立つ部門です。モジュールと呼ばれる収束装置を携行し、現場での即応処置を行います。

部門詳細
Division-02

支援部門

主任務: 収束部門の後方支援 / 運用管制 / 医療・心理ケア。 想定脅威度: 低〜中。

現地オペレーションの調整やデータ解析、海蝕員の帰還後ケアを担当します。状況に応じて現場への同行支援も実施されます。

部門詳細
Division-03

工作部門

主任務: 海蝕実体の解析 / モジュール開発 / 機関設備の新造。 想定脅威度: 変動。

回収された海蝕実体や残滓を検証し、その特性をモジュールへ転用します。部室・消耗品・小物類の製作も兼任します。

部門詳細
Division-04

外事部門

主任務: 外界および階底次元との交渉 / 情報工作 / 隠蔽・広報対応。 想定脅威度: 中。

行政・報道機関・階底次元住民との折衝やメディア操作を行います。スカウトや経理に類似した業務を担当することもあります。

部門詳細
Division-05

港湾部

主任務: 海域監視 / 埠頭封鎖 / 境界流入防止。 想定脅威度: 中〜高。

境界の入り口となる土地や施設を24時間監視し、不根ふねや海蝕実体の侵入を阻止します。

部門詳細

機密規約抜粋

海蝕現象に関する一切の情報は、機関外部への口頭・文書・電子的送信を禁じられています。 違反した場合、記憶処理および階底次元送致を含む懲戒が行われます。

本セクションは一般海蝕員向けの抜粋版です。完全版規約は権限レベル3以上の海蝕員のみ閲覧可能です。
